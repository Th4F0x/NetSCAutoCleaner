## Скрипт очистки
-----------------------------------------------------------
Скрипт на python который чистит папки которые указаны в config.json
-----------------------------------------------------------
Пример config.json
Для виндос указывается путь с двумя "\\" в линукс как обычный путь

```
{
  "targets": {
    "Загрузки": "C:\\Users\\User\\Downloads",
    "Видео": "/home/user/Видео",
    "Кастомная папка": "D:\\Temp\\Cache"
  }
}
```
----------------------------------------------------
Данный скрипт преднозначен как один из скриптов для NetNexus
